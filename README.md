# rtmp-camera-save
Saves RTSP feeds from IP cameras

## Installation
```sh
npm i
```

## Building and running
```sh
npm run build
```

## Running
```sh
node <path/to/rtmp-camera-save> # Outside rtmp-camera-save dir
npm start                       # Within rtmp-camera-save dir
```

## Development
```sh
npm run init # Optional, installs development helpers
npm run dev
```
